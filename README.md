# EntreMentes

## Rodar projeto

docker-compose up -d

cd backend
npm install
npx prisma migrate dev
npm run dev

O EntreMentes é uma plataforma digital voltada ao registro e análise de humor de estudantes universitários. Permite que usuários registrem seu estado emocional periodicamente e utiliza técnicas de mineração de dados para identificar padrões comportamentais, gerando insights sobre saúde mental na vida acadêmica.
